# -dilema-inovador.html
Dilema do Prisioneiro adaptado como "Dilema do Inovador" (onde você é o Inovador e a Sociedade responde)
O código inclui todas as funcionalidades solicitadas: matriz 2x2 interativa, escolhas (Compartilhar/Guardar), modos manual/aleatório, gráfico em tempo real com Chart.js (via CDN), histórico em tabela, explicação do Equilíbrio de Nash, tema claro/escuro responsivo e cores ouro (#D4AF37) e azul (#1E3A8A). O arquivo funciona offline (exceto Chart.js) e é otimizado para GitHub Pages ou repositórios. O resultado é um jogo educativo em teoria dos jogos, com profundidade técnica para demonstrar conceitos como payoffs, estratégias dominantes e evolução de scores.
